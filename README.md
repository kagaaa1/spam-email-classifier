# Spam Email Classifier

A machine learning project to classify emails as **Spam** or **Not Spam**.

## Objective
Build a model that detects spam emails using text processing and ML.

## Structure
- data/ → datasets  
- src/ → code (preprocessing, models, etc.)  
- outputs/ → results  
- main.py → final integration
